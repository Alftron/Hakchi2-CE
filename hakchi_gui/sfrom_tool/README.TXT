In order to use SFROM Tool with hakchi2 CE, you need to:
	
- visit /u/DarkAkuma's SFROM Tool website: http://darkakuma.z-net.us/p/sfromtool.html
- download the latest version of his tool;
- install the package in this folder;
- enable use of SFROM Tool in Settings -> SFROM Tool -> Enable;
- subsequent import of snes games will be run through the tool.
